###A Chrome Extension for all you copy pasters###

More to come... Currently this can only be compiled with `grunt build` and loaded as an unpacked extension.

---

####Installation####

1. npm install
2. bundle install
3. bower install

---

####Development####

`grunt debug` allows for watching of files and starts a server reached at http://localhost:9000

---

####Build####

`grunt build` creates a dist directory from which you can load an unpacked extension.
